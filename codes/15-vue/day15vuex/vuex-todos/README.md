# 初始化项目

1. 安装: `npm install -g @vue/cli`
2. 创建项目: `vue create vuex-todos`
3. 运行: `npm run serve`

# 把 todos 模板拿过来

1. 拷贝模板中 <section>
2. 拷贝 node_modules > base.css/index.css
3. 再 main.js 中引入 css

# 组件化改造

1. 创建 todoHeader.vue，把头部标签代码拷贝进去
2. 引入组件
3. 注册组件使用组件

# 配置 vuex

1. 安装：`npm i vuex`
2. 创建文件 store.js
   > router/router.js
   > store/store.js

- 引入
- 实例化 store
- 导出

3. 挂载到 vue 实例上
4. 准备数据
